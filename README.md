### ✨ About the Evolution

**WebNova** has evolved. This repository houses the current iteration of the agency's platform, moving beyond a standard landing page to a fully interactive web application.

We shifted to **Vite** for blazing-fast performance and integrated **Framer Motion** to breathe life into every interaction. The design philosophy remains strictly minimalist, but the user experience is now rich with modern effects, smooth transitions, and dynamic content.

> *It's not just about information; it's about the feel of the navigation.*

---

### 🎨 Key Features & Experience

- **⚡ Instant Performance**: Migrated to **Vite** for optimized building and lightning-fast HMR (Hot Module Replacement).
- **🎭 Motion-First Design**: Heavily utilizes **Framer Motion** for complex layout animations and scroll-triggered effects.
- **💎 Utility-Based Styling**: Crafted with **TailwindCSS** and `tailwindcss-animate` for precise, maintainable, and responsive designs.
- **🧭 Seamless Navigation**: Client-side routing managed by **React Router DOM** for a SPA (Single Page Application) feel.
- **📸 Interactive Galleries**: Implemented **Swiper** for touch-friendly, fluid sliders and carousels.
- **🔥 Serverless Power**: Connected to **Firebase** for real-time database capabilities and secure authentication.
